    Web Analytics Insights

    Consent-based website analytics — all outbound clicks the site tracks (storefronts AND community links), in relation to total site traffic. The view for judging the website itself, not the media buying.

    Two data worlds on this page: the sections above use ad-platform attribution (paid traffic, storefront conversions). This section uses consent-based web analytics — it counts only consenting visitors, includes ALL outbound destinations (storefronts, community, socials), and reports with up to 2 days delay. The two will never match exactly; compare trends, not absolutes.
    Site Traffic Consent-based users, sessions and page views per day.
    Outbound Clicks by Destination & Market Every outbound destination the site links to (stores, community, socials) × top markets.
    Click any column to sort.
    User Density Map Consent-based site users per country (color intensity) — the fastest read on where
    the website audience sits and which regions remain untouched. Hover a country for detail.
    Site Performance by Market Consent-based users, sessions, page views and outbound-click rate per market.
